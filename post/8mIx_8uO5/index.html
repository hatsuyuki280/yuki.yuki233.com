【迁移成功】本站已成功从WordPress顺利迁移到了Gridea！

2021-04-08 杂谈 公告

本站已经顺利完成了基础的功能的迁移，因此特意发了这篇文章庆祝一下（顺便当作新站点的HelloWord文章）

其实昨天也写过同样标题的一篇文章，不过由于顺便发现了一个 Gridea 的Bug。。。（其实就是保存失败了）。。总之就重写了。。。

那些都先扔一边，总之先跟大家分享一下这次迁移的各种变化好啦quq：

【致歉】由于这次的迁移，从 WordPress 迁移到了仅有静态界面的博客系统，因此将会彻底丢失以前的依赖数据库的全部内容及所有需要后端支持的功能。包括：

  • 订阅用户的注册信息
  • 所有页面的评论
  • 基于邮件的本站的更新通知

此外，原有的博文的永久链接也将失效，如果有进行署名转载的话，也许还需要更新一下新的连接。对于上文提到的数据库的维护将在本站域名从备用的测试域名 https://bbgp.yuki233.com/ 迁移到blog的原始域名 https://yuki.yuki233.com/ 完成的同时，会停止对新写入内容的同步。所有数据将会保留3个月，虽然咱觉得大家应该没有什么重要的数据保留在咱的站点上，不过出于信息安全的考虑，在未来三个月内将可以通过备用的测试域： https://bbgp.yuki233.com/ 访问原始站点并在登陆后下载相关个人数据，3个月后本站将在没有任何通知的情况下对所有用户数据进行安全的删除处理，届时将会永久性的删除原站以及相关的所有数据库。咱会在原站和测试站的域名交换的同时，向全部订阅用户使用邮件通知站点更新的相关信息。
考虑到原有的各种评论均无法迁移到新的评论系统，咱预计将会使用归档的方式，除：

  • 友链申请信息
  • 已被标记为垃圾评论的信息
  • 已删除的信息
  • 仅有表情符号或颜文字的信息

外，会对评论内容由咱进行人工脱敏，遮挡：

  • 用户邮箱地址
  • 用户个人网站
  • 部分用户名

后，按页整理为YAML风格的纯文本文档，在今后陆续随博文迁移到新站时，添付在文章内。今后如果需要移除这些页面上的旧评论内容，可以直接联系咱进行删除。

【变化】从即日起，本站的用户注册功能将会予以关闭，原有订阅用户将会永久失去对所属账户的访问权限，并失去原有的邮件通知功能，如需继续订阅本站的更新，欢迎访问本站所属的 存储库 通过Github的 Watch 功能进行邮件订阅。同时本站的RSS地址也将变更为 https://yuki.yuki233.com/atom.xml ，以便有需要的用户使用自己喜爱的RSS订阅工具进行订阅。

【变化】从即日起，本站的评论系统将会迁移到更适合静态站点的 Gittalk。因此评论时将会需要使用大家的 Github 账号进行登录。为了方便大家大家理解并使用 Gittalk，稍后将会发布一个简易的Github注册教程。届时将会粗略的介绍 Github 以及 大家的评论会如何被保存。如果有需要的话可以一并去阅读。

【变化】新站点变更了界面的样式，一方面是原有样式表迁移成本比较高，另一方面是咱的水平有限，短时间内可能无法完成这么大量的对接工作，因此暂时选用了结构相似的内置样式表，未来如果有机会的话，会考虑将原有样式表重写成新站点也可以使用的格式。
此外，由于后端变动，暂时禁用了原有的飘雪特效和2233娘，预计将会在未来的版本中逐步添加适配。

【新增】新站将会新增位于页脚或侧边栏的访客计数器小工具。目前预计采用的，是由 玩水 dalao制作的 Moe-counter 。考虑到迁移成本，暂时还没有在本地进行独立部署，因此在这里感谢 玩水 dalao的项目。
*未来有计划将计数器迁移到本地。

【保持】本站将会继续一如既往的咕咕咕和水文章，考虑到咱的技能方面确实太弱，短时间之内也确实想不出来更多可以更新的内容。。。因此。。更新速度依然随缘。还请大家多多谅解。。。
此外，本站将会一如既往的使用由 DogeCloud 友情提供的 CDN图床 服务作为本站的部分图片资源及小文件的的分发及缓存。站点的 CDN 也一如既往的使用 Cloudflare 。因此连接性上应该不会和原来有太大的体验落差。相对的，新站点采用全静态的站点文件配合 GitPage 和 Cloudflare Page 进行存储和分发，相信可以在一定程度上，进一步提高使用体验。

【变化】由于本站的迁移，关于用户数据的隐私声明等，将会在未来进行更新。关于相关声明将在发布同时生效，如有相关需求，届时请注意更新内容。

【关于原有博文】由于原站的文章数量不多不少正好在手动迁移有些麻烦，但有不值得写一个一次性的迁移脚本的程度。。。因此本站的迁移将会异步完成。在与正式域名交换后，咱将会逐步重写原有博文。因此短时间内会有大部分博文的链接失效，如有需求，可以适当的配合 Google 的站点缓存进行审阅。

【关于友链】由于新站迁移，友链信息需要重新录入，友链页面也需要重新构思。因此咱决定在此期间，手动重新审阅全部友链，并按照预计会采用的新规则对友链内容进行重新评估，届时会对不符合要求的友链及友链位置将会有所变动的各位站长，会在咱能力范围内进行沟通。关于新的友链交换规范，将会在未来进行发布。如有相关需求，届时请注意关注本站的更新。在新的友链交换规范发布前，本站将会暂停新增友链的申请及邀请。如果造成不便，还请谅解。

【关于多语言】众所周知，咱位于日本，且英语非常烂。目前有计划会对站点做多语言支持，但主要也只会面向日语进行本地化，其他语言目前可能还得让大家依赖Google翻译。

【Aff相关】目前虽然有接到一些 Aff 相关的邀请，在咱的再三考虑下，出于访问体验的考虑，依然没有选择在站内嵌入广告。不过未来预计会给咱的blog新增 小黄油 推荐、 VPS 推荐等内容，但应该不会很频繁。尤其是 小黄油推荐 和 本子推荐 之类的，咱不会直接分享内容，但是会在页面中适量的配一些评论的截图予以介绍。aff链接 由 DL Site 提供，DL Site 咱认为是一个比较具有知名度的 电子版 小黄油 同人志 的贩卖站，经常还会有一系列的大额优惠活动，同时也支持基于点卡的支付宝付款，所以即便没有信用卡，使用起来应该也不会有困难。如果有喜欢的创作者在使用 DL Site 的话，不妨去试试支持一下她们的商品的销量吧。
另外说是准备开始搞 Aff，咱也会以自己的喜好进行分享，毕竟不是正经广告，所以并没有想靠这个创造什么收益。大家也不需要为了咱去点击他们，按大家的喜好来就好。*（因为只是点击的话并不会给咱产生返利

【预计会新增的项目】咱经过了很长一段时间的思考，未来将有可能丰富咱的博文内容，包括但不限于：

  • 咱自己写的小脚本的介绍与分享
  • 小黄油、ASMR、同人志（主要是萝莉本）的推荐（上文也提到了，如果有人购买这些推荐的话会产生一定的返利）
  • 更多的日常吐槽（但是应该不会很频繁，同时考虑到咱的信息不如大家的群里人灵通，很多时候大概也只是互联网信息的最后消费者。。。所以大概是没办法成为大家的信源的）
  • 由于最近开始学习画画了，未来看情况可能会开始投稿一些咱的练笔作品。
  • 由于咱还是学生，考虑到写作业就已经很累了，说不定会和大家分享一些咱的作业。。。虽然应该没办法成为教程。。但如果可以在一些方面给没有学过的大家提供一些参考也是好的。
  • 日语类，虽然大家好像对日语类教程没有多大兴趣，不过考虑到咱也就只会日语了，所以还是会考虑分享一些日语、日本相关的小知识。
  • 最后是教程类，主要是咱来回摸索之后和大家分享一下踩过的坑和总结出来的东西。

如果有更多的想法了，咱也会直接开始搞。同时，以上内容并不保证一定都会开始搞，毕竟本站的宗旨之一就是咕咕咕！

总之期待在新的站点和大家见面了quq！
