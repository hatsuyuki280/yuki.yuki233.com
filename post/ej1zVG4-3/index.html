# 什么是五十音

首先需要明确一点的是，虽然五十音叫做五十音，但并不是因为由五十个发音构成。所谓五十音是指的日语的发音准则。

细致的来说，五十音有“元音”“半元音”“辅音”“拨音”“促音”“拗音”“长音”，“清音”“浊音”“半浊音”和“平假名”“片假名”三种分类方式，其中“元音”“复元音”“辅音”“拨音”“促音”“拗音”“清音”“浊音”“半浊音”“长音”为发音的类型分类的，平片假名为写法的分类。

此处本来有图片的，但是还没迁移完成

以上的内容的详细介绍将在后续内容中进行。

 

在这里说名这一点的目的是为了给大家提前打破五十音图是说日语有五十个发音的错误印象。总的来说如果要计数量的话，五十音图包含的发音量为

50(5段10行)-5(わ行的3个和や行的2个)+1(ん拨音)+25(4行5段的浊音+1行5段的半浊音)+45(15行1段*ゃゅょ的拗音)

共计116个非促音非长音非拨音组合的发音，如果换算成假名，共计232组假名，如果算上促音+长音+拨音的常见组合，将会更多，这里就不完整统计了。总的来说只要在这一小节知道五十音图并不只是说日语有50个发音就对了。

五十音图

其中 かさたは行有浊化，分别为がざだば行，は行有半浊化，为ぱ行。浊化半浊化等均未出现在上述表格中，可以从这里获取pdf格式的文档

读音之类的问题可以参考网络上的五十音的录音，这里就不多赘述了。但是要注意的是す的发音更偏向于汉语拼音的si；すつしち的发音和浊化后的发音也是比较特殊的，或者说可能发音会完全一致，所以如果可能的话，遇到这些发音的组合记得着重记忆一下正确的假名哦~~~比如つづく（つずく）的发音就是完全一致的，但是括号中的后者就是错误的，所以一定要重点确认哦~
